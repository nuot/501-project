Collecting Method and Source

In order to successfully perform the analysis, we decided to use three data sets.

The first data set looking at the top 116 hard-working cities in America containing their rankings and scores. Then, we used Yelp API to gather data about the theaters such as theater ID, the name of the theaters, ratings, and the number of reviews it has in those cities.

Lastly, based on the theater ID, we gather the top 3 reviews of that theater and the review rating.

Data Cleaning

After running the Yelp API, we decided to do some general cleaning:

• Remove the empty value (We do not think empty value offers any useful information to our analysis)
• Remove invalid data such as non-numerical zip code
• Remove outliers using the statistical method (we decided to use two standard deviations from the medium, and we decided not to use our LOF since we have an abundance of categorical variables which leads to less-ideal results)

Then, we got a dataset that contains close to 15,000 entries of theater information. We decided to eliminate those theaters that do not have its location in the list of 116 hard-working cities caused by the insufficient radius we used. Then, we decided to remove all the duplicate rows and the theaters that have 0 reviews, which leads us to the dataset that we use to analyze our data science question.
